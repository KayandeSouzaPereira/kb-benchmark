# Per-plan API rate tiers

FREE: 60 req/min. PRO: 600 req/min. ENTERPRISE: custom, negotiated. This
applies to the public REST API only, not to the outbound webhook limit (see
[outbound-webhook-rate-limit.md](outbound-webhook-rate-limit.md), which is a
flat 10/hour for every plan).
